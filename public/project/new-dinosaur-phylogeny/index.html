New dinosaur phylogeny

August 6, 2017
Network Analysis

数ヶ月前に、恐竜の系統樹を大きく修正する必要があると主張する論文が発表された。

リンク：https://www.nature.com/nature/journal/v543/n7646/full/nature21700.html

これが元の系統樹。

最節約法で作成されたという。

かなり大胆な系統で、関係者は皆驚いたようだ。

当然、多くの人が著者に連絡し、生データを提供してもらって、それぞれ再分析に挑んでいるようだ。

そのデータがなぜか、僕のところにも回ってきた。

僕は系統樹分析に割と詳しいので、入手したデータで早速ネイバーネットを作ってみた。

これは僕が作成したネットワーク。

うまくクロップ出来ず、解像度がかなり落ちてしまっているが、

枝先に同じ色をつけているので、比較するに支障はない。

元の論文では紫色と青色が一つのグループを形成していて、それが更に緑色と結合している。

この点が定説の系統樹と対立する点である。

しかし。

ネットワークをみてみたら、紫色と青色が綺麗に結合しているかというと、そうではない。

更に、青色と緑色も明白に重なっていて、尚且つレティキュレーションも多い。

これが何を意味するのかというと、元の論文の系統樹は、かなりの不確実性を内包しているものの、

それがちゃんと検証されていないということ。

そのため、今回の結果が統計的なアーティファクトである可能性を十分に排除していない。

僕が思うには、今回の結果は恐らく「最節約法」に依存する結果になっている。

しかし、まぁ、僕は今回の論文に関しては、かなり懐疑的ではあるが、

従来の知見や定説に思いっきりチャレンジする研究姿勢は学びたい。
